## Build a template

`POST /templates/build`

Starts a template build in the background and returns immediately. Poll `GET /templates/{template_id}` to check when it's ready.

**Request body**

Pass exactly one of `image_ref` (pull from registry — no host builder
needed) or `dockerfile` (build locally — requires docker/podman/buildah).

```json
// Pull a pre-built image
{
  "name": "my-python-env",
  "image_ref": "python:3.11-slim",
  "rootfs_size_mb": 500,
  "inject_flint": true
}

// Build a Dockerfile locally
{
  "name": "my-python-env",
  "dockerfile": "FROM python:3.11-slim\nRUN pip install numpy pandas",
  "rootfs_size_mb": 500
}
```

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `name` | string | required | Human-readable name |
| `image_ref` | string | — | OCI image reference. Mutually exclusive with `dockerfile`. |
| `dockerfile` | string | — | Full Dockerfile contents. Mutually exclusive with `image_ref`. Returns `400` if no builder is installed. |
| `rootfs_size_mb` | int | `500` | Rootfs image size in MB |
| `inject_flint` | bool | `true` | Copy `flintd` and `init-net.sh` into the rootfs. Set to `false` for pre-built Flint images. |

Images referenced via `image_ref` are pulled with
[crane](https://github.com/google/go-containerregistry) (no Docker
daemon). Builds from `dockerfile` use whichever builder is available on
the host (`podman`, `buildah`, or `docker`).

**Response** `200`

```json
{
  "template_id": "tpl_abc123",
  "status": "building"
}
```

## List templates

`GET /templates`

**Response** `200`

```json
{
  "templates": [
    {"template_id": "default",   "name": "Default",        "status": "ready"},
    {"template_id": "tpl_abc123","name": "my-python-env",  "status": "ready"}
  ]
}
```

---

## Get template

`GET /templates/{template_id}`

**Response** `200` or `404`

```json
{
  "template": {
    "template_id": "tpl_abc123",
    "name": "my-python-env",
    "status": "ready"
  }
}
```

Status values: `"building"`, `"ready"`, `"failed"`

---

## Delete a template

`DELETE /templates/{template_id}`

You can't delete the `"default"` template.

**Response** `200`

```json
{"ok": true}
```

**Response** `400` - if you try to delete `"default"`

**Response** `404` - template not found
